# Miles Kenneth Napilan Portfolio

A modern portfolio website built with Next.js and styled with Tailwind CSS.

## Run locally
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
npm run start
```

This project already uses the default Next.js build pipeline. You do not need any custom `nextjsbuild` setup for GitHub or Vercel.

## Deploy to Vercel
1. Push this folder to GitHub
2. Import the repository in Vercel
3. Keep the default framework preset as `Next.js`
4. Deploy
